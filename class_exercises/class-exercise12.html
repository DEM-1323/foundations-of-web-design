Class Exercise 12:

Practicing CSS Animations and Transitions

Goal:

Create a webpage using animations and transitions to create interesting effects.

Instructions:

Download starter_code12.zip:

  • Unzip the folder to get started.

Setup your Files:

  • Open your “Week 6” folder and drag the files from the “starter_code” folder into your “Week 6” folder.
  • Open your “CE12” folder and edit the “CE12.html” file.

Add Internal CSS to your HTML File:

  • Use the <style> tag and add the following CSS styles:
    • Add a transition property to your .box styles that puts a 0.5s duration and ease on the transform and background-color properties.
    • Add a transform: scale() property and change the background-color to #2ecc71 on your #transition-box:hover styles.
    • Add an animation property to #animation-box assigning the moveBox animation playing infinitely.

Viewing Your Web Page:

  1. Run Your Project with Live Server:
    • Right-click on your “CE12.html” file and select “Open with Live Server”.
    • This will open your web page in your default browser.
    • You should now see your custom-styled web page.
  2. Run Your Project with Live Preview:
    • Right-click on your “CE12.html” file and select “Show Preview”.
    • This will open your web page in a tab in VS Code next to your HTML file.
