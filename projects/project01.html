Project 1:

Creating a Themed Website

Goal:

Create a themed website with:
  • 3 webpages
  • Headings, paragraphs, and images
  • Navigation links between pages
  • External links
  • Lists (ordered or unordered)

Instructions:

Create a New HTML File:

  • In your “Week 1” folder, create a new folder named “Project1”.
  • Open your “Project1” folder and create an HTML file named “index.html”.

Website Requirements:

  • Each page should include headings, paragraphs, and images
    • You must use <h1> - <h3> tags
    • Include 1 image per page
  • Use the <div> tag to make distinct sections on at least one page
  • Use the <span> tag to have colored text on the page
  • Use the <a> tag to link to your other HTML pages, and have at least one external link
  • Include the use of lists somewhere on your website
    • Ordered or unordered lists

Create the Additional Pages:

  • In your “Project1” folder, create two more HTML files named “page1.html” and “page2.html”.

Set Up the Basic HTML Structure:

  • For each of your HTML files (index.html, page1.html, page2.html), type the following basic HTML structure:
<!DOCTYPE html>
<html>
<head>
  <title>Themed Webpage</title>
</head>
<body>
  <div class="navigation">
      <a href="index.html">Home</a>
      <a href="page1.html">Page 1</a>
      <a href="page2.html">Page 2</a>
  </div>
  <div class="content">
      <!-- Content for each page goes here -->
  </div>
</body>
</html>

Adding Content to Each Page:

  • Start filling out each page with content related to your theme. Include headings, paragraphs, and images as described in the requirements.
  • Ensure you use <div> for distinct sections and <span> for colored text within your paragraphs.
  • Include links to other pages within the navigation div and at least one external link in the content.

Viewing Your Web Page:

  1. Run Your Project with Live Server:
    • Right-click on your “index.html” file and select “Open with Live Server”.
    • This will open your web page in your default browser.
    • Navigate through your site using the link in the navigation div.
  2. Run Your Project with Live Preview:
    • Right-click on your “index.html” file and select “Show Preview”.
    • This will open your web page in a tab in VS Code next to your HTML file.

Submission:

  • Ensure all your files are correctly linked and functional.
  • Compress your “Project1” folder into a zip file.
  • Submit your zipped “Project1” folder with all your HTML files here: Submit Project 1
