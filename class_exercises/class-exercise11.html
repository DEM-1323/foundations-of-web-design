Class Exercise 11:

Practicing CSS Keyframes

Goal:

Create a simple animation using CSS keyframes to understand how to define and apply animations to HTML elements.

Instructions:

Download starter_code11.zip:

  • Unzip the folder to get started.

Setup your Files:

  • In your “Foundations of Web Design” folder, create a new folder named “Week 6”.
  • Open your “Week 6” folder and drag the files from the “starter_code” folder into your “Week 6” folder.
  • Open your “CE11” folder and edit the “CE11.html” file.

Add Internal CSS to your HTML File:

  • The .animated-box class applies the animation named colorChange which lasts for 4 seconds, runs infinitely, and alternates directions.
  • The @keyframes rule defines the stages of the animation, changing the background color from blue to green to yellow.
  • Use the <style> tag and add the following CSS styles:
    • Add an additional animation stage at 25% and 75%.
    • Include a background-color change at those stages.
    • Use the transform: rotate() property at each stage.
    • The animated-box should rotate 90 degrees at each stage.

Viewing Your Web Page:

  1. Run Your Project with Live Server:
    • Right-click on your “CE11.html” file and select “Open with Live Server”.
    • This will open your web page in your default browser.
    • You should now see your custom-styled web page.
  2. Run Your Project with Live Preview:
    • Right-click on your “CE11.html” file and select “Show Preview”.
    • This will open your web page in a tab in VS Code next to your HTML file.
