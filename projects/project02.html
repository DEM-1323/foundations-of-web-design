Project 2:

Styling Your Themed Website: Example Site

Goal:

Take your website from Project 1 and style it using an external CSS file. Use basic CSS properties like background color, color, font size, padding, and margins to change the look of your existing site.

Instructions:

Use Your Project 1 Files:

  • Start with the files and structure you created in Project 1.
  • Make a copy of your “Project1” folder and rename it to “Project2”.

Create a New CSS File:

  • In your “Project2” folder, create a CSS file named “styles.css”.

Link the CSS File to Your HTML Files:

  • Open each of your HTML files (index.html, page1.html, and page2.html).
  • Add the following line within the <head> section to link your CSS file: 
    
<link rel="stylesheet" href="styles.css">

Modify the CSS File:

  • Open your styles.css file.
  • Use the following CSS properties to style your website:
    • Background Color: Change the background color of different sections.
    • Color: Change the text color.
    • Font Size: Adjust the size of headings and paragraphs.
    • Padding: Add space inside elements.
    • Margins: Add space outside elements.

Viewing Your Web Page:

  1. Run Your Project with Live Server:
    • Right-click on your “index.html” file and select “Open with Live Server”.
    • This will open your web page in your default browser.
    • Navigate through your site using the link in the navigation div.
  2. Run Your Project with Live Preview:
    • Right-click on your “index.html” file and select “Show Preview”.
    • This will open your web page in a tab in VS Code next to your HTML file.

Submission:

  • Ensure all your files are correctly linked and functional.
  • Compress your “Project2” folder into a zip file.
  • Submit your zipped “Project2” folder with all your HTML files here: Submit Project 2
